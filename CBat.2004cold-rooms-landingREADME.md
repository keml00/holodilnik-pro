# Холодильные камеры - Premium Landing

Современный premium-лендинг для компании по производству и монтажу холодильных камер.

## Технологии

- **Next.js 16** - React фреймворк
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **Framer Motion** - анимации
- **Lucide React** - иконки

## Структура

- **Header** - sticky навигация с CTA
- **Hero** - главный экран с формой заявки
- **Stats** - статистика компании
- **Catalog** - каталог холодильных камер (6 типов)
- **Advantages** - преимущества (6 блоков)
- **Process** - этапы работы (5 шагов)
- **Cases** - реализованные проекты
- **Reviews** - отзывы клиентов
- **FAQ** - частые вопросы (accordion)
- **FinalCTA** - финальный призыв к действию + footer

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

## Сборка

```bash
npm run build
npm start
```
